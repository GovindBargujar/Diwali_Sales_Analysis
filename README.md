# Project Overview:-
  Conducted a comprehensive data analysis project aimed at optimizing Diwali sales strategies from a given dataset, focusing on sales patterns, customer behavior, and marketing effectiveness during the Diwali season.

# Data Collection & Management:-
  Gathered and cleaned sales data from various sources including transaction databases, etc.
  Utilized SQL to query and extract relevant data, ensuring accuracy and completeness.

# Data Analysis & Visualization:-
  Analyzed sales trends and customer demographics using statistical methods and data visualization tools.
  Developed predictive models to forecast sales trends and identify key drivers of sales performance during the Diwali season.
